# 介绍

## 为什么建立这个站点？

好记性不如烂笔头。建立这个个人文档站点的初衷是：

1. **沉淀知识**：把零散的技术笔记整理成体系化的文档
2. **方便检索**：比翻聊天记录、笔记 App 更高效
3. **分享交流**：公开的内容可以帮助有同样问题的人

## 技术栈

本站基于 [VitePress](https://vitepress.dev/) 构建，部署在 GitHub Pages 上。

- **构建工具**：VitePress
- **托管平台**：GitHub Pages
- **自动部署**：GitHub Actions
